# Tarefa EBAC
tarefa focada pra aprimorar os conhecimentos em estrutura de repetição
utilizando o "FOR" e "IF" e a classe Scanner

# Scanner
Usando para solitar o nome e um numero maior que 0.

#IF
Usado para verificar a quantidade de letras tem no nome do usuário e se o número escolhido foi positivo.

#FOR
Usado para contar o número escolhido pelo usuário de forma crescente e decrescente, além de repedir o nome caso a quantidade de letras passe de 6.
